# 🧠 Quote Generator CLI App

A simple and fun command-line Python app that displays inspirational quotes!  
You can fetch random quotes from an online API or from a local file, and even save your favorite ones.

---

## 🚀 Features

- ✅ Fetch random quotes from an API
- ✅ Fallback to local quotes if offline
- ✅ Save favorite quotes to a text file
- ✅ Beginner-friendly and lightweight
